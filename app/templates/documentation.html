Benutzer Dokumentation

Übersicht

Diese Applikation erlaubt unseren internen Operations Admins, einen User gezielt von verschiedenen Security Policies temporär zu exkludieren.

User temporär von Security Policy Exkludieren

  1. Navigiere zu Exclusion Manager.
  2. Gib die E-Mail des zu exkludierenden User ein.
  3. Wähle die Security Policy aus, von der der Benutzer exkludiert werden soll.
  4. Wähle das Ablaufdatum.
  5. Klicke "Zur Exclusion-Liste hinzufügen".

Bestehende Exkludierungen anzeigen

  1. Navigiere zu Exclusion Manager.
  2. Klicke auf "Datenbank auslesen".
  3. Jetzt werden alle User mit einer aktiven exkludierung angezeigt.
Anmerkung:
  • Ausgelaufene exkludierungen werden nicht angeziegt.
  • Ausgelaufene exkludierungen können in der Datenbank nachgeschaut werden.
